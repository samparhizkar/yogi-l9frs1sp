# Theme

The `theme` tokens components is responsible to merge and exposes all the Yogi Design Token in one consistent object.

This _complete theme_ is then ready to be injected in the [Chakra-UI's `extendTheme`](https://chakra-ui.com/docs/theming/customize-theme#customizing-theme-tokens) theming function:

```jsx
import React from 'react';
import { ChakraProvider, extendTheme } from '@chakra-ui/react';
import { theme } from '~/theme';

// Extend default Chakra theme
const fullTheme = extendTheme(theme);

// Usage
<ChakraProvider theme={fullTheme}>
  {/***
    Your components tree
  */}
</ChakraProvider>;
```
